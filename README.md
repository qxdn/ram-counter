# 采用RAM实现计数器设计

## 要求

用一个10*8的双口RAM完成10个8位计数器，计数器的初值分别为1~10，时钟频率为1MHz，计数器计数频率为1Hz.用FPGA开发板上的按键作为计数器计数值的输出选择控制，用开发板8个LED作为选择计数器的计数值输出。
